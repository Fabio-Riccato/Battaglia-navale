# Client - Battaglia Navale

Applicazione Flutter lato giocatore.

## Requisiti
- Flutter SDK installato
- Dart incluso in Flutter

## Avvio
```bash
flutter pub get
flutter run
```

## Verifiche
```bash
flutter analyze
flutter test
```

## Note
Questo progetto è parte del monorepo `Battaglia-navale` insieme a `server/`.
Assicurati che l'ambiente Flutter sia compatibile anche con i vincoli SDK del server.
